# Employee Attrition Prediction

A Machine Learning project that predicts employee attrition using the IBM HR Analytics Employee Attrition dataset.

## Project Overview

Employee attrition is an important challenge for organizations. This project explores employee data, performs preprocessing and visualization, and builds a Machine Learning model to predict whether an employee is likely to leave the company.

## Dataset

IBM HR Analytics Employee Attrition Dataset

The dataset contains employee information such as:

- Age
- Department
- Job Role
- Monthly Income
- Years at Company
- Work-Life Balance
- Overtime
- Job Satisfaction
- Education
- Business Travel
- Attrition Status

## Project Workflow

1. Data Loading
2. Data Cleaning
3. Exploratory Data Analysis (EDA)
4. Feature Encoding
5. Data Visualization
6. Model Training
7. Model Evaluation
8. Model Saving

## Visualizations

The project includes several charts inside the **charts/** folder:

- Attrition Distribution
- Department-wise Attrition
- Job Role Attrition
- Monthly Income Distribution
- Correlation Heatmap
- Confusion Matrix

## Machine Learning Model

Model Used:

- Random Forest Classifier

Evaluation Metrics:

- Accuracy
- Precision
- Recall
- F1 Score
- Confusion Matrix
